回溯算法思想能解决矩阵、棋盘、排列、组合、子集等问题
经典题目：如全排列.java, 括号生成.java, 组合.java, 子集.java
需要回顾：带"_需要回顾"的字样需要回顾

核心思路：选择列表 -> 路径（迭代选择列表，做出选择，递归，取消选择） -> 终止条件

核心模板：
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return

    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
